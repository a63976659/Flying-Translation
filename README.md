# 🌐 ComfyUI Flying Translation (飞行翻译)

# [![Bilibili](https://img.shields.io/badge/bilibili-猪的飞行梦-00A1D6?logo=bilibili&logoColor=white)](https://space.bilibili.com/2114638644)

**ComfyUI 专属现代化、多模态本地化辅助引擎** | **开箱即用的云端/本地双擎翻译工作站**

无需繁琐的 JSON 手动对齐，在 ComfyUI 侧边栏打造你的专属翻译与补丁合并流水线！

支持一键截图提取、静态语法树扫描，完美适配多国语言，极致保护您的翻译进度！



-----

## ✨ 核心亮点 (Highlights)

  * 🚀 **云端本地双擎驱动**：无缝切换本地显卡算力（支持 Qwen2.5/MiniCPM 等）与专属云端矩阵。算力耗尽或显存不足时，云端排队系统为你无感接力！
  * 👁️ **多模态视觉直翻 (VLM)**：首创截图翻译提取！支持拖拽或读取剪贴板节点截图，大模型逆向解析提取结构。独家“扫描主键校准”技术，利用 AST 智能将截图显示名替换为底层真实类名。
  * 🧠 **智能 AST 静态解析**：拒绝动态 import 炸环境。纯静态语法树降维解析代码，完美向下兼容旧版 `INPUT_TYPES` 与新版 `define_schema` 节点架构。
  * 🛡️ **高阶 JSON 编辑器与安全合流**：内置带“红绿灯状态机”的高级代码审查面板。一键查重、自动修复缺失逗号、排除已翻译项，并支持物理硬盘级的防抖存档与撤销/重做。
  * 📂 **多语言路由矩阵**：深度联动 `ComfyUI-Chinese-Translation` 基础多语言框架，自动识别并精准覆写至对应的语言包目录（如 `zh-CN`, `ja-JP`, `ru-RU` 等）。未安装基础框架亦有本地输出目录完美兜底。

## 📦 安装说明 (Installation)

**1. 克隆仓库**
进入你的 ComfyUI 插件目录（`custom_nodes`）：

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/a63976659/Flying-Translation.git
```

**2. 安装极简依赖**
在你的 ComfyUI Python 虚拟环境中执行（秋叶启动器无需进行这一步）：

```bash
pip install aiohttp huggingface_hub
```

*(就这么简单！重启 ComfyUI，点击侧边栏的 飞行翻译 即可开启硬核汉化体验。)*

## 🎮 使用说明 (User Guide)

### 💻 1. 源码提取 (AST 静态扫描)

自动扫描 `custom_nodes` 目录下的所有插件，也可点击“外部装载”直读本地任意文件夹。

  * **流式切片翻译**：大文件自动分割为 15 组数据的块进行翻译，防止长连接超时报错。
  * **合并与追加**：当目标目录已存在同名 JSON 时，系统会智能唤起“合并冲突模式”，自动将旧数据追加在下方供您审查。

### 👁️ 2. 视觉解析 (VLM 逆向提取)

遇到极其复杂的动态节点？截图解决！

  * **精准提取**：复制节点图片并在拖拽区点击“读取剪贴板截图”。
  * **三步走校准**：1. 核对大模型提取的插件名；2. 点击智能匹配确认归属 JSON；3. 点击“扫描源码并替换”，系统会自动修正键名防爆红。

### 🧩 3. 补丁融合 (极速内存合并)

专为插件更新迭代设计。

  * **无缝覆盖**：分别选中“主文件 A (旧翻译)”和“补丁文件 B (新提取)”，点击极速内存合并。
  * **安全覆写**：合并后的数据将在冲突编辑器中呈现，确认无误后强制覆写落盘。

  * 所有云端 API 请求均通过 `https://zhiwei666-flying-translatio-api.hf.space` 进行安全转发。
  * 本地模式下，你的代码与截图数据绝不出网，完全在本地显卡显存中完成闭环推理！
